## 什么是ES6

一般将ES2015之后的版本，如ES2015、ES2016、ES2017、ES2018称为ES6。ES6泛指“下一代 JavaScript 语言”。每年的6月份，TC39 委员会都会发布ECMAScript的新版标准。
